# EPA FMS - Admin Dashboard

Modern React.js admin dashboard for EPA Freight Management System.

## Features

- ✅ Material-UI (MUI) components
- ✅ Professional UI/UX design
- ✅ Admin authentication
- ✅ Dashboard with KPIs and charts
- ✅ User management
- ✅ Shipment management
- ✅ Compliance & Customs
- ✅ Reports & Analytics
- ✅ Inventory management
- ✅ Billing & Invoicing
- ✅ Full API integration

## Getting Started

### Installation

```bash
cd epa-dashboard
npm install
```

### Development

```bash
npm run dev
```

The dashboard will be available at http://localhost:3000

### Build for Production

```bash
npm run build
```
